1. 1. What is the difference between invoking and creating a function?

    2. Invoking a function just means calling on, to use, a function that already exists. Creating a function, like the word "create" implies, means creating a new function.

  2. 2. Explain the syntax regarding semicolons and code blocks.

    3. We kinda use the semicolon as a punctuation mark in javascript. Code blocks are where we write the code we want to see executed inside loops and functions.

  3. 3. What happens when we try to store values from a function when it doesn't return any?

    4. "Undefined" maybe?

  4. 4. How do we separate parameters from each other when creating a function?

    5. We use commas to separate parameters from each other when creating a function.

  5. 5. Explain how parameter names are related during the invocation and creation of a function.

    6. When creating a function you use a substitute name, for example "aNumber", and when you invoke the function you write an actual number, which then is used inside function instead of "aNumber"

  6. 6. Explain why a parameter is undefined when a function call is missing a parameter.

    7. Maybe because the program is trying to look for a parameter that's supposed to be there, but isn't.

  7. 7. Explain what happens when we use recursive functions.

    8. When using a recursive function it will repeat until it reaches a set condition, kinda like loops.
