1. Where should you use constants, and where should you use variables? Can they all be constants?

    2. Man bruker konstanter på ting som ikke skal endres, f.eks fødselsår. Variabler brukes på ting som kan endres, f.eks alder. Det kan være lurt å bruke konstanter der det er mulig, og så bruke variabler på resten. Alt kan ikke være konstanter, fordi det er er alltid noe som må kunne endres

  2. How do parentheses affect the arithmetic outcome of an expression?

    3. Parantes påvirker rekkefølgen på regnestykket, så man kan få forskjellig svar med og uten parantes.

  3. How do we limit floating-point numbers to a given number of decimals?

    4. Man kan bruke metoden toFixed(antall desimaler du vil ha). Man kan også bruke metoden toString(), men den har jeg ikke prøvd selv.

  4. How can we compare two floating-point values to see if they are equal (is 12.199999 the same as 12.2)?

    5. Man kan bruke == for å se om de er like. Man kan visst også bruke alligatortegn.

  5. What do comparison operators return, and how can we check if they are FALSE or TRUE?

    6. Når man skriver inn == eller >< så får man tilbake true eller false.

  6. How does the Math.random() function work, and what can it be used for?

    7. Denne funksjonen vil gi et tilfeldig tall innenfor de verdiene du har satt. Det kan f.eks brukes hvis du skal lage et terningspill.

  7. What is modulus (%), and when do we need this operator in mathematical operations?

    8. Modulus brukes for å finne det som er igjen etter å ha delt noe. F.eks 5 delt på 2 er 2, og da er det 1 igjen.
